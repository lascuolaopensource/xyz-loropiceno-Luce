Configurazione archivio

Imposta il token GitHub per salvare automaticamente ogni esportazione nella cartella files/ del repository. Il token resta solo su questo browser — non viene mai inviato altrove se non a api.github.com.

Token GitHub (PAT)

Come creare il token

  1. Vai su github.com/settings/personal-access-tokens e crea un token fine-grained.
  2. Imposta Repository access su Only select repositories e scegli LucaTommy/Luce.
  3. Sotto Repository permissions imposta Contents su Read and write.
  4. Crea il token, copialo e incollalo qui sopra, poi clicca Salva.
