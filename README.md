# Luce 💡

**Luce** è un tool per progettare elementi grafici e sviluppato durante il laboratorio co-progettuale **"Sapere, fare, diventare" XYZ** organizzato da **La Scuola Open Source**, svoltosi a Loro Piceno dal 3 al 9 Maggio 2026.

Il progetto nasce all'interno dell'ecosistema identitario del workshop, integrando dinamiche di visual identity, tecnologie web e la restituzione dei dati attraverso il font dedicato **Q-Sci**.

**[Q-Sci](https://github.com/lascuolaopensource/xyz-loropiceno-font-Q-Sci)**


### 📁 Struttura della Repository

```text
├── .github/workflows/    # Automazione e CI/CD
├── css/                  # Fogli di stile per il layout e la UI della dashboard
├── js/                   # Logica dell'applicazione (gestione flussi e dati)
├── files/                # Asset e dati di supporto
├── index.html            # Landing page principale del tool
├── setup.html            # Pagina di configurazione iniziale del sistema
├── dashboard.html        # Pannello di controllo e monitoraggio dati in tempo reale
├── save_log.php          # Script backend per il salvataggio dei log di stato
├── dashboard_api.php     # Endpoint API interno per la comunicazione dei dati
├── Q-Sci.otf             # File sorgente del sistema tipografico Q-Sci
├── QSci-Regular.otf      # Variante Regular del carattere di progetto
├── QSci-Black.otf        # Variante Black per accenti e titolazioni di impatto
└── Logo.svg              # Identità visiva e vettoriale del tool
